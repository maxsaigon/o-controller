# UI/UX Review Notes

This folder is for Codex-owned visual QA notes and screenshots.

Do not store service logs or protocol fixtures here.

Expected future files:

- `desktop-normal.md`
- `desktop-offline.md`
- `desktop-pending.md`
- `desktop-metadata.md`
- screenshots captured during visual review

## Review Protocol

1. Run the desktop app against mock service.
2. Capture the menu bar popover in normal, offline, pending, and metadata states.
3. Check text fitting, control spacing, disabled states, and dark/light mode.
4. Record issues and fixes in this folder.
